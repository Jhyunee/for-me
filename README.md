## 💭 개발 동기
목표 달성에 따른 **나를 위한 후회 없는 소비의 적정선**을 찾아가는 서비스<br><br>
**`“갓생“`** **`“셀프 기프팅“`**
* MZ세대의 중심 트렌드 - **‘나를 위한 투자’, ‘행복한 오늘에 집중’**
* 경제관념을 무너뜨리는 2030 **리볼빙 이슈**
* 위 키워드를 종합하여, 오늘을 계획하고 목표를 향한 동기부여를 제공하는 편리한 서비스 기획.
<br><br><br>

## 💸 기대 효과
개인의 행복을 위한 후회 없는 소비, 건강한 ‘미코노미’의 적정선을 찾아가는 서비스 제공
* 주 사용자층 예상: MZ세대 및 2030세대
* 계획된 보상과 성취를 통해 과소비 방지 및 건강한 소비습관, 생산성 있는 습관 구축에 도움
* 추후 아동 대상 경제관념 교육 취지로 소비층 확대
<br><br><br>

### 🌟 주요 기능
* **Daily Checklist** : 일자별 목표 할일 작성 및 이동 가능, 완료 체크
* **달성 통계 제공** :  자동 추출된 체크리스트 카테고리에 따른 년/월/주 별 달성 통계 제공
* **마이페이지** : '노력금' 설정 및 수정, 누적 노력금 확인, 개인정보 수정
* **커뮤니티** : 타 유저들의 체크리스트 랜덤 열람, 달성 노력금 비교

## 🏃🏻‍♀ How to Run
**▶️ [AI prediction model](https://github.com/Jhyunee/for-me-classification)**
<br><br>

**▶️ Backend Server**
```
./application.yml
  address : #YOUR_IP_ADDRESS
  password : #YOUR_MYSQL_PASSWORD
Run SpringDeveloperApplication.java
```
<br>

**▶️ Frontend Client**
```
for-me/Forme-front/
  $ npm install
  $ npm start
```
